🔑 Como obter sua Chave API do D4Sign
① CRIAR CONTA

Acesse secure.d4sign.com.br/criar e cadastre-se com o e-mail do escritório

② SOLICITAR ATIVAÇÃO

Envie e-mail para [email protected] pedindo ativação da API para sua conta

③ PEGAR O TOKEN

Faça login na D4Sign → clique no menu (canto superior direito) → opção "Dev (API)"

④ UUID DO COFRE

Na mesma tela "Dev (API)" ou em Cofres, copie o UUID da pasta onde os documentos serão salvos
